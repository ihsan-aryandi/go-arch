# Golang Architecture

Struktur projek Golang berskala menengah ke atas yang dibuat sesuai standarisasi dan rekomendasi dari Golang.

---

## Folder Utama

Folder utama dalam struktur projek ini ada 3 yaitu :
### 1. Folder "cmd"
Gampangnya folder ini digunakan untuk menjalankan program atau aplikasi Anda seperti `go run main.go`. Tetapi, di folder ini Anda juga bisa menambahkan beberapa file main.go dengan tujuan yang berbeda-beda. Contohnya seperti migration, scheduler, dan lain-lain.
